## Getting Started

1. Clone the repo
```
git clone https://github.com/imondok03/Nice-Gadgets.git
cd phone-catalog
```

2. Install dependencies
```
npm install
```

3. Start the project
```
npm start
```


## Features

  - Responsive Design: Fully optimized for all screen sizes and devices.

  - Favorites & Cart: Add products to favorites or the shopping cart, with automatic total price calculation.

  - Product Filtering: Filter products by capacity and color directly from the product card.

  - Sorting: Sort products based on criterias: year, price, alphabeticaly.

  - Pagination: Browse through large product lists, select the number of items per page, with pages updating accordingly.

  - Sticky Header: The header remains visible while scrolling for easy navigation.

  - Scroll to Top Button: Quickly return to the top of the page with one click.

  - Product Sliders ‚Äî Featured sliders, allowing users to quickly browse trending or recently added products.
